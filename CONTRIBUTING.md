# Contributing to EduTutor AI

感谢您对 EduTutor AI 的关注！本项目的主文档位于 `edututor-ai/` 目录下。

## 📖 快速指引

请参阅 [edututor-ai/CONTRIBUTING.md](edututor-ai/CONTRIBUTING.md) 获取完整的贡献指南。

## 🚀 快速上手

1. Fork 本仓库
2. 创建特性分支：`git checkout -b feature/my-feature`
3. 提交改动：`git commit -am 'feat: add my feature'`
4. 推送到分支：`git push origin feature/my-feature`
5. 提交 Pull Request

## 📋 开发须知

- 项目基于 **Spring AI 1.0.0-M6** + **JDK 21**
- 所有 PR 需通过 CI 检查
- 代码风格遵循 Google Java Style
- 提交信息遵循 Conventional Commits
